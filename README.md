## About the Project

This repository contains my data engineering projects completed as part of the MIT xPRO Professional Certificate in Data Engineering. The projects demonstrate proficiency in ETL pipelines, database design, data analysis, and API development.

**Key Projects:**
- **A Model To Predict Housing Prices**: A multiple linear regression model to predict sale prices of houses using Python
- **MRTS Time Series Analysis**: ETL pipeline processing 30 years of U.S. retail sales data
- **Restaurant API**: REST API built with Strapi and PostgreSQL
- **Data Visualization**: Interactive dashboards and analytics
